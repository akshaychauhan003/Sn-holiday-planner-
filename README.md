## Architecture
This is a vanilla web application built without complex build steps or node modules, focusing on high performance and clean document flow.
- **HTML5**: Semantic structure
- **CSS3**: Custom properties, Flexbox/Grid layouts, mobile-first responsive design, and glassmorphism UI elements.
- **JavaScript (Vanilla)**: DOM manipulation, IntersectionObserver for video lazy-loading, and responsive mobile behaviors.

## Setup Instructions
1. Clone or download the repository.
2. Open `index.html` in any modern web browser.
3. No `npm` or build process is required.
